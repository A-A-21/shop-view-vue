# shop-view-vue
Небольшое приложение с витриной магазина, где можно добавлять в корзину. Vue 3 Composition API, Vuex.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
